# Учетные Записи

## Описание проекта

Это веб-приложение для управления учетными записями пользователей, разработанное с использованием современных технологий Vue 3, TypeScript и Element Plus. Приложение позволяет пользователям создавать, редактировать и удалять учетные записи с возможностью добавления тегов и выбора типа аутентификации (локальная или LDAP).

## Технологии

- **Vue 3** - Прогрессивный фреймворк JavaScript для создания пользовательских интерфейсов
- **TypeScript** - Язык программирования, надстройка над JavaScript, добавляющая статическую типизацию
- **Element Plus** - Компонентная библиотека для Vue 3, обеспечивающая богатый набор UI-компонентов
- **Pinia** - Магазин состояний для Vue.js, замена Vuex
- **Vite** - Современный инструмент сборки, обеспечивающий быструю загрузку приложения
- **Vue DevTools** - Инструменты разработчика для отладки приложений Vue
- **Pinia Plugin Persistence** - Плагин для сохранения состояния Pinia в localStorage

## Функциональность

### Основные возможности:
- Создание новых учетных записей
- Редактирование существующих учетных записей
- Удаление учетных записей
- Добавление тегов к учетным записям
- Выбор типа аутентификации (локальный или LDAP)
- Валидация вводимых данных
- Сохранение данных между сеансами (через localStorage)

### Типы учетных записей:
- **Локальные** - Традиционные учетные записи с логином и паролем
- **LDAP** - Учетные записи, использующие внешнюю систему аутентификации LDAP (без хранения пароля)

### Валидация:
- Логин: длина от 1 до 100 символов
- Пароль: длина от 1 до 100 символов (только для локальных учетных записей)
- Теги: общая длина не более 50 символов
- Отображение визуальных индикаторов ошибок

## Запуск проекта

### Предварительные требования
- Node.js версии ^20.19.0 или >=22.12.0
- npm или yarn

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```
Приложение будет доступно по адресу http://localhost:5173

### Сборка для продакшена
```bash
npm run build
```

### Просмотр собранного приложения
```bash
npm run preview
```

## Структура данных

Каждая учетная запись имеет следующую структуру:
```typescript
interface AccountModel {
  id: number;              // Уникальный идентификатор
  tags: string[];          // Массив тегов
  recordType: 'local' | 'ldap'; // Тип аутентификации
  login: string;           // Логин
  password: string | null; // Пароль (null для LDAP)
  errorFields: string[];   // Поля с ошибками валидации
}
```

## Компоненты

### App.vue
Главный компонент приложения, который управляет основным макетом и включает компоненты Hint и Accounts.

### Accounts.vue
Компонент, реализующий таблицу учетных записей с возможностью редактирования полей, добавления тегов и удаления записей.

### Hint.vue
Простой компонент для отображения подсказок пользователю.

## Хранилища (Stores)

### accounts.ts
Хранилище Pinia для управления данными учетных записей, включая:
- Добавление новых учетных записей
- Удаление учетных записей
- Обновление полей учетных записей
- Сброс паролей для LDAP-записей
- Управление ошибками валидации
- Персистентное хранение данных

## Особенности реализации
